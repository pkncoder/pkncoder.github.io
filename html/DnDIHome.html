Welcome to dndinfinity

This is an interactive site that can show stats for monsters, spells, classes, etc.



Info about this site

Some spells, monsters, classes, etc will not show up.

I apologize as this data is not accessable due to the only avalible data is the free rulebook, a.k.a. the DnD SRD.

This is still in development, and will not be done for a while (meant to keep me busy).



Extra bugs

There are some bugs that will happen

One notable one is the data not showing up from formatting issues when data is requested from the API

There may or may not still be more formatting errors, however most if not all are taken care of

If any extra bugs are found then please report them to https://github.com/pkncoder/pkncoder.github.io/issues


Api data

This site uses the DnD 5th Edition Api

Link to their site: https://www.dnd5eapi.co

If this goes down then indeed some things won't work

Thanks to the developers of DnD 5th Edition Api for letting people like me to be able to utilize it
