Element Search

Element Search Webapp v2


How does this work?

There are three different ways to search for an Element.

With use of a JSON file, your search is found and data is givin back to you.

If you wish to find and use this JSON file, it is in the recources tab in the navbar


Name

Go to the 'Search With Name' tab on the top of the site

Input any name for an element

*This will default to Hydrogen if an error occors, or the element doesn't exist*


Protons

Go to the 'Search With Protons' tab on the top of the site

Input any Atomic Number (1-119)

This give the information for that element


Search With Symbol

Go to the 'Search With Symbol' tab on top of the site

Input any atomic symbol for an element

This will default to Hydrogen if it doesn't exsist or an error occors



Api

If you don't know, an api is basicly a way to connect 2 computers together over the internet

The api that I used was very slow to update. Thankfully, I was able to use it and turn it into a JSON file

The JSON file that I use is inside the recources tab in the navbar.
